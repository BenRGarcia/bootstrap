Terminal Shortcuts

  • Move cursor between words with: option + right arrow or option + left arrow
  • Delete the word left of the cursor with: control + w
  • Move cursor to the far right of the line with: control + e
  • Move cursor to the far left of the line with: control + a
  • Delete the entire line left of the cursor with: control + u
  • Delete the entire line right of the cursor with: control + k

Make an Alias some next-level stuff...

Do you find yourself using the same terminal commands over and over again? And perhaps are they super long or difficult to remember?

Consider if you had to run this command all the time: git log --pretty=format:"%h - %an, %ar : %s"

That's really tedious to type, so... Let's make up a command name of our own (an alias)!

  • In your root directory, you'll add aliases with $ vim .bash_profile
  • And the syntax looks like this:
    • alias prettylog='git log --pretty=format:"%h - %an, %ar : %s"'
  • Then, refresh the bash shell environment with $ source ~/.bash_profile
  • And now, when you give the command $ prettylog,
    terminal actually runs $ git log --pretty=format:"%h - %an, %ar : %s"
